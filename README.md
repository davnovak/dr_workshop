# Single-cell dimensionality reduction workshop

The `data` directory contains (pre-processed) Drop-Seq data from the *Farrell et al. (2018)* study.
The data was obtained from the Single Cell Portal using accession code *SCP162*.

**CITATION**:

* Farrell, J. A., Wang, Y., Riesenfeld, S. J., Shekhar, K., Regev, A., & Schier, A. F. (6 2018). *Single-cell reconstruction of developmental trajectories during zebrafish embryogenesis*. Science, 360. doi:10.1126/science.aar3131
